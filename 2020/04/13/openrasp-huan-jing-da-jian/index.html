OpenRASP环境搭建

suanve
2020-04-13
安全
openrasp, rasp

https://rasp.baidu.com/doc/
访问该地址看了下基础的文档

安装

https://github.com/baidu-security/openrasp-testcases/releases
下载对应的测试用例

启动一个php的容器
docker run -d --name rasp -p 80:80 -v $(pwd):/var/www/html --link mysql suanve/php:7-apache

进入docker容器后下载对应的程序。

默认不拦截，需要手动改为拦截

添加拦截

vim /opt/rasp/plugins/official.js

修改all_log为false，这样就可以让大多数规则变成拦截

测试

在web目录下写一个phpinfo

确定出现类似的内容

打开主页，检查返回包有没有OpenRASP。

都有就代表安装完成。

随便打开一个测试用例

可以直接复制curl命令，测试发现拦截。

直接访问也是拦截。
